# Microsoft Fabric and Azure Databricks OneLake Federation

## Overview
This project demonstrates the integration between Microsoft Fabric and Azure Databricks to leverage OneLake for efficient data federation.

## Features
- **Data Federation:** Seamlessly integrate data from multiple sources.
- **Scalability:** Efficiently scale with the power of Azure Databricks.
- **Performance Optimization:** Enhance data processing speeds with Microsoft Fabric.

## Requirements
- Microsoft Fabric account
- Azure Databricks workspace
- Access to OneLake

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/<username>/Microsoft-Fabric-and-Azure-Databricks-OneLake-Federation.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Microsoft-Fabric-and-Azure-Databricks-OneLake-Federation
   ```
3. Install necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To start using the project:
- Configure the settings in the `config.yaml` file.
- Run the application:
   ```bash
   python app.py
   ```
